## CadastroContato

Este projeto foi desenvolvido com o objetivo de praticar os conceitos de programação por meio da criação de um sistema simples de cadastro de contatos.

A aplicação permite cadastrar, consultar, atualizar e remover contatos, facilitando o gerenciamento das informações de cada pessoa cadastrada. O projeto foi desenvolvido como exercício para aprimorar a lógica de programação, a organização do código e a manipulação de estruturas de dados.

## Funcionalidades

- Cadastro de contatos
- Consulta de contatos cadastrados
- Atualização de informações
- Remoção de contatos
- Gerenciamento da lista de contatos
  
## Tecnologias

- Python
  
## Como utilizar

1. Faça o download ou clone este repositório.
   
2. Abra o projeto em uma IDE compatível com Python, como Visual Studio Code ou PyCharm.
   
3. Execute o arquivo principal para utilizar o sistema de cadastro de contatos.
